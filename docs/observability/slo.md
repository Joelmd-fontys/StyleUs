# Draft Service-Level Objectives

## Availability
- Target 99.5% availability for public API endpoints once live.

## Latency
- P95 response time below 400 ms for styling recommendations in production.

## Quality
- Automated regression suite coverage at or above 80% prior to releases.

## Monitoring Plan
- Define initial dashboards and alert thresholds before production launch.
- Integrate observability tooling decisions in a future infrastructure ADR.
