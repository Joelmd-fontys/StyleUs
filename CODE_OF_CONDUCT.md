# Code of Conduct

This project adheres to the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct.html).

- Foster an inclusive, harassment-free community.
- Use respectful language and assume positive intent.
- Report unacceptable behavior to the maintainers at contact@styleus.dev.

Maintainers will enforce this code in good faith and may moderate contributions or participation that violate these expectations.
